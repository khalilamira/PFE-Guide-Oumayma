The "Add New Pentest" feature allows users to initiate a new penetration test (pentest) within the Igider platform. This functionality is essential for identifying and addressing security vulnerabilities in a structured and efficient manner. How to Use Accessing the Feature: Navigate to the "Pentest" section in the left-hand sidebar menu. Click on "Add New Pentest" to open the pentest creation form. Form Fields: Title: Enter a unique and descriptive title for the pentest. This will help in identifying and managing different pentests within the platform. Description (Optional): Provide a brief description of the pentest. This field is optional but can be useful for additional context and details about the test. Pentest Scope: Select the scope of the pentest from the dropdown menu. The scope defines the boundaries and extent of the pentest, specifying what assets and areas will be tested. Pentest Intensity: Choose the intensity level of the pentest. This could include options such as Low, Medium, or High, indicating the thoroughness and depth of the testing. Select Assets: Select the assets that will be included in the pentest. This ensures that the specific assets to be tested are clearly identified and targeted. Initiating the Pentest: Once all the necessary fields are filled out, click the "Add New Pentest" button to initiate the pentest. If you need to cancel the operation, click the "Cancel" button to discard any entered information and return to the previous screen.
